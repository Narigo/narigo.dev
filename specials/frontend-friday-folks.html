Frontend Friday Folks

In VirtualCoffee.io, we started a group that meets every Friday to work on a CSSBattle.dev puzzle. I think there hasn't been a single time where I haven't learnt a new CSS trick.

There was enough interest to actually start a second session per week now. I'm sharing solutions I've done here, but I'm open to share others as well, if someone wants to submit anything.
